---
使用 `Echarts` 内置字体，如下：
```jsx
const options = {
    ...,
    textStyle: {
        fontFamily: "monospace" //  'serif' , 'monospace', 'Arial', 'Courier New', 'Microsoft YaHei', ...
    },
    ...
}
```
更多请访问 [textStyle.fontFamily](https://echarts.apache.org/zh/option.html#textStyle.fontFamily)
